# Citizen Flavor - Supported Functionalities

The SKILLAB platform provides a comprehensive suite of functionalities designed to bridge the gap between labor market demand and skill supply. These functionalities are delivered through the orchestration of platform microservices, ensuring that users receive evidence-based insights derived from large-scale data processing.

## Common Platform Services

All versions of the SKILLAB platform include a foundational set of services related to User Management. Powered by a custom-developed service, these common functionalities include secure user registration, authentication, and profile management.

All flavors also have access to core labor market analytics:

- Demand Analysis: Using a large repository of job advertisements, users can access descriptive and exploratory analytics. This includes Hierarchical Cumulative Voting (HCV) to identify the most critical skills for specific occupations, TURF analysis to uncover optimal skill combinations, and Co-occurrence Networks to visualize relationships between professional requirements.
- Labor Market Forecasting: The platform provides predictive modelling for occupational demand, enabling users to visualize the evolution of specific roles and assess the mortality risk (obsolescence) of specific skills within the current market.
- Supply Analysis: The platform processes five distinct data streams: EU Profiles, Short Courses, EU Knowledge Units (KUs), EU Syllabi, and EU Policy documents. Users can apply the same analytical lenses (HCV, TURF, and Forecasting) to these supply sources to understand the current educational and professional landscape.

## Citizen-Specific Functionalities

The Citizen flavor is designed for individual career pathing and upskilling.

- Skill Profile Orchestration: Users can define their current skill sets and set target occupations.
- Gap Analysis and Course Recommendation: The platform compares the user's profile with the requirements of a target occupation, identifies skill gaps, and recommends specific educational courses to bridge those deficiencies.
- Personalized Career Guidance: Based on the user's existing competencies, the platform provides recommendations for alternative occupations that match the user's skills.
